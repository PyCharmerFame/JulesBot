# JulesBot

## Инструкция по запуску проекта

Этот проект использует GitHub Actions для автоматического создания задач в Jules на основе сообщений из Telegram.

### 1. Настройка секретов (Secrets)

Для работы проекта необходимо добавить следующие секреты в настройки репозитория (Settings -> Secrets and variables -> Actions):

*   `TG_BOT_TOKEN`: Токен вашего Telegram-бота (получить у @BotFather).
*   `TG_CHAT_ID`: ID чата в Telegram, из которого бот будет читать задачи.
*   `JULES_API_KEY`: API ключ для доступа к Jules API.

### 2. Запуск

Проект работает автоматически через GitHub Actions.
Существует два способа запуска:
*   **По расписанию:** Проверка новых сообщений происходит каждые 5 минут.
*   **Ручной запуск:** Вы можете запустить workflow "Telegram to Jules" вручную на вкладке "Actions" (workflow_dispatch).

### 3. Использование

В указанном Telegram-чате отправьте сообщение, начинающееся с `/task `, например:
`/task добавь в README инструкцию запуска проекта`

При следующем запуске GitHub Action бот прочитает это сообщение и автоматически создаст задачу (Pull Request) в Jules.

### 4. Интерактивные кнопки бота

Для более удобного управления и мониторинга процессов в боте реализовано меню с интерактивными кнопками:
* **📊 Статус процесса**: Выводит информацию о последних запущенных GitHub Actions с указанием процента выполнения и статуса (например, ⏳ В очереди, 🔄 Выполняется, ✅ Успешно).
* **📂 Активные PR**: Показывает список открытых Pull Requests в вашем репозитории.
* **❓ Помощь**: Выводит справочную информацию по доступным командам.
